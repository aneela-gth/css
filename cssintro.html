CSS (Cascading Style Sheets):


CSS is a stylesheet language used to describe how the content written in HTML should look on a webpage.
It controls the design, colors, fonts, layout, spacing, and overall presentation of a website.
HTMLdefines the structure (what is on the page).
CSS defines the style (how it looks).

Example:

1.Without CSS → A webpage looks plain with only text and images.
2.With CSS → The same webpage can have colors, backgrounds, proper alignment, animations, and a professional look.
In short: CSS makes web pages attractive, consistent, and user-friendly.
